## 🔧 Customization

### Adjust Physics
```javascript
// In quantum-particle-cart.js, edit CONFIG:
const CONFIG = {
  particleRadius: 60,      // Size
  gravity: 0.15,           // Falling speed
  friction: 0.98,          // Motion damping
  bounce: 0.6,             // Bounciness
  centerForce: 0.02,       // Pull to center
  maxVelocity: 8           // Speed limit
};
```

### Change Colors
Edit gradient colors in `quantum-particle-cart.js`:
```javascript
gradient.addColorStop(0, `rgba(0, 240, 255, ...)`);    // Cyan
gradient.addColorStop(0.5, `rgba(255, 0, 170, ...)`);  // Magenta
gradient.addColorStop(1, `rgba(0, 255, 136, ...)`);    // Lime
```
