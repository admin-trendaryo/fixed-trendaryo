## 🔧 CUSTOMIZATION OPTIONS

### Adjust Physics
```javascript
// Slower motion
gravity: 0.05, friction: 0.95, centerForce: 0.01

// Faster motion
gravity: 0.25, friction: 0.99, centerForce: 0.03

// Bouncier
bounce: 0.8, friction: 0.97
```

### Change Colors
Edit gradient colors in `quantum-particle-cart.js`:
```javascript
gradient.addColorStop(0, `rgba(0, 240, 255, ...)`);    // Cyan
gradient.addColorStop(0.5, `rgba(255, 0, 170, ...)`);  // Magenta
gradient.addColorStop(1, `rgba(0, 255, 136, ...)`);    // Lime
```

### Adjust Particle Size
```javascript
particleRadius: 60  // Increase for bigger particles
```
