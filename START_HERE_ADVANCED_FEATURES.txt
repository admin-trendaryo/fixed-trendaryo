💾 DATA STORAGE
================================================================================

All data stored in browser localStorage:

  trendaryo_analytics      - Analytics events
  trendaryo_search_history - Search history
  trendaryo_reviews        - Product reviews
  trendaryo_wishlist       - Wishlist items
  cache_*                  - Cached data
  trendaryo_errors         - Error logs
  trendaryo_csrf_token     - CSRF token

Storage Limit: ~5-10MB per domain
Auto-Cleanup: Old data removed when quota exceeded

🚀 USAGE EXAMPLES
================================================================================

TRACK PRODUCT VIEW
──────────────────
Analytics.trackProductView(123, 'Wireless Headphones');

TRACK ADD TO CART
─────────────────
Analytics.trackAddToCart(123, 2, 149.99);

SEARCH PRODUCTS
───────────────
const results = SearchEngine.search('headphones', {
  minPrice: 50,
  maxPrice: 200
});

VALIDATE FORM
─────────────
const { isValid, errors } = Validator.validateForm(formData, {
  email: { required: true, email: true },
  password: { required: true, password: true }
});

ADD REVIEW
──────────
ReviewsManager.addReview(123, {
  author: 'John Doe',
  rating: 5,
  title: 'Great!',
  text: 'Excellent product'
});

ADD TO WISHLIST
───────────────
WishlistManager.addItem(productObject);

CACHE DATA
──────────
cache.set('products', productsArray);
const products = cache.get('products');

EXPORT ANALYTICS
────────────────
Analytics.exportAnalytics(); // Downloads CSV
