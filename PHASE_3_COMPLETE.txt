================================================================================
🐛 TROUBLESHOOTING
================================================================================

ISSUE: "Order not found"
SOLUTION: Verify order ID, check authentication, verify order ownership

ISSUE: "Cannot cancel order"
SOLUTION: Check order status, verify ownership, check if already shipped

ISSUE: "Tracking info not available"
SOLUTION: Order may not have shipped, tracking number not assigned

ISSUE: "API returns 401"
SOLUTION: Verify token is valid, check token expiration, refresh token

ISSUE: "Export not working"
SOLUTION: Check browser console, verify orders exist, check permissions
