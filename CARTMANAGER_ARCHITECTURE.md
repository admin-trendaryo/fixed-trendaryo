│\n└─────────────────────────────────────────────────────────────────┘\n\n┌─────────────────────────────────────────────────────────────────┐\n│ 1. Custom Event Listener                                        │\n│    window.addEventListener('cartUpdated', updateCartCountBadge) │\n│    ✓ Fires immediately when cart is modified                    │\n│    ✓ Fastest response time (< 1ms)                              │\n└─────────────────────────────────────────────────────────────────┘\n                              │\n                              ▼\n┌─────────────────────────────────────────────────────────────────┐\n│ 2. Storage Event Listener                                       │\n│    window.addEventListener('storage', updateCartCountBadge)     │\n│    ✓ Fires when localStorage changes in other tabs              │\n│    ✓ Enables cross-tab synchronization                          │\n│    ✓ Response time: < 5ms                                       │\n└─────────────────────────────────────────────────────────────────┘\n                              │\n                              ▼\n┌─────────────────────────────────────────────────────────────────┐\n│ 3. DOMContentLoaded Listener                                    │\n│    document.addEventListener('DOMContentLoaded',                │\n│                              updateCartCountBadge)              │\n│    ✓ Fires when page first loads                                │\n│    ✓ Ensures badge is initialized                               │\n│    ✓ Restores cart count from localStorage                      │\n└─────────────────────────────────────────────────────────────────┘\n                              │\n                              ▼\n┌─────────────────────────────────────────────────────────────────┐\n│ 4. Manual Call on Page Load                                     │\n│    window.addEventListener('load', () => {                      │\n│        updateCartCountBadge();                                  │\n│    });                                                          │\n│    ✓ Backup initialization                                      │\n│    ✓ Ensures badge is always updated                            │\n└─────────────────────────────────────────────────────────────────┘\n```\n\n## API Method Call Hierarchy\n\n```\n┌─────────────────────────────────────────────────────────────────┐\n│
